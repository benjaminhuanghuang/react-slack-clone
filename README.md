## Steps
1. Create react app 
```
  npx create-react-app slack-clone
```

2. Setup firebase
- create project
- Install firebase SDK
```
  npm i firebase
```
- Copy config to project


3. Install Material UI
```
  npm i @material-ui/core @material-ui/icons -S 
```
